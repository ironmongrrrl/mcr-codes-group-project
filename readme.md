Welcome to the holding bay for Manchester Codes Sept 2020 Cohort final project ideas! 

<h2>To add an idea:</h2>
1. Hit up Lucy Ironmonger on Slack and request to be a collaborator.<br>
2. Open up terminal, and go: **git clone git@github.com:ironmongrrrl/mcr-codes-group-project.git**<br>
3. Add to the readme, in format below.<br>
4. Push it up using **git push origin main**, adding a commit message with your full name in.<br>
5. If you get any issues pushing up (e.g. a merge conflict), try:<br>
- **git pull** (you may need to check the readme and 'Accept both changes')<br>
- then try **git merge --continue --no-edit** (this completes the process of merging latest changes from the remote repository to your local branch along with the merge conflict fix you've just applied.)<br>
- <a href="https://www.manchestercodes.com/platform/module/programming-foundations/git-intro">This link</a> is quite good if you get stuck.
